Task 2: Prototyping Approach & Screen Mockups

1.	Selected Prototyping Approach:
We chose: (Evolutionary Prototyping) for our project.
2.	Justification: We selected Evolutionary Prototyping for the SAMS project. Since our system handles interactive tasks like searching events and booking seats, this approach lets us start with basic screen sketches and gradually update them into the final working interface. This saves time and avoids starting from scratch.
